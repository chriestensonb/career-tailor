# Job Description: Platform Automation Engineer

Northstar Tools is hiring a Platform Automation Engineer to improve internal
developer workflows. The team maintains CI pipelines, release tooling, and
observability dashboards for product engineers.

## Responsibilities

- Build Python command line tools for release and deployment automation.
- Improve GitHub Actions workflows for testing, packaging, and publishing.
- Partner with application teams to reduce manual operational work.
- Add observability for background jobs, APIs, and developer tooling.
- Write clear documentation for repeatable engineering workflows.

## Requirements

- 3+ years of Python experience.
- Experience with CI/CD systems, GitHub Actions, and package publishing.
- Comfort with APIs, structured data, and command line interfaces.
- Strong written communication and a bias toward practical automation.
